Project Overview

In this screencast, I walk viewers through the main features of the application while explaining how it can support teaching and learning. The recording combines live on-screen actions with clear narration to make each step easy to follow.

Reflection

Creating this screencast helped me think more carefully about pacing, clarity, and visual focus. I learned how to structure demonstrations so that viewers are not overwhelmed, and how to highlight the most important elements of the interface to support understanding.
