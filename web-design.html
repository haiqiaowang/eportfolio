Web Design Projects

These projects explore interactive web design using HTML, CSS, and JavaScript. Both quizzes focus on creating clear layouts, intuitive interactions, and playful learning experiences.

Quiz 1 • Baking-Themed Quiz

This quiz invites users to answer multiple-choice questions in a baking theme. I focused on clean typography, friendly color choices, and clear feedback so that learners can easily follow their progress.

  • Built with HTML, CSS, and JavaScript
  • Multiple-choice interactions with instant feedback
  • Designed to be simple, fun, and easy to navigate
View Quiz 1

Quiz 2 • Drag-and-Drop Classification

This drag-and-drop quiz lets users sort items into categories by moving them on the screen. The goal was to create a more dynamic interaction that encourages experimentation and active learning.

  • Drag-and-drop interaction using JavaScript
  • Visual grouping to support classification and comparison
  • Simple, responsive layout for different screen sizes
View Quiz 2

Reflection

Through these web design projects, I practiced planning page structure, writing clean and readable HTML, and using CSS to create a consistent visual language across different pages. JavaScript allowed me to add interactivity so that quizzes feel more like playful learning experiences than static assessments.

Working on both a traditional quiz and a drag-and-drop activity also helped me think about how different interaction patterns can support different kinds of thinking—such as recall, comparison, and classification.
